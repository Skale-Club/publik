KDP Cover Dimensions Formula (from RESEARCH.md):
- Cover Width = Bleed (0.125") + Back Cover + Spine + Front Cover + Bleed (0.125")
- Cover Height = Bleed (0.125") + Trim Height + Bleed (0.125")
- Spine Width (B&W white) = Page Count × 0.002252"
- Spine Width (B&W cream) = Page Count × 0.0025"

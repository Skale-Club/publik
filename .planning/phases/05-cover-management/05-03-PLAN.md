# Validation Logic Requirements

KDP Cover Specifications (from RESEARCH.md):
- Resolution: 300 DPI minimum (600 DPI recommended)
- Color mode: RGB (KDP converts to CMYK)
- File format: JPEG, PNG, TIFF, or PDF
- File size: 50MB maximum
- Bleed: 0.125" on all sides
- Minimum dimensions: Calculated from trim size + spine + bleed at 300 DPI

Validation should check:
1. Front cover: minimum width/height based on trim size, page count, paper type
2. Back cover image: minimum dimensions (usually smaller than full cover)
3. File format: must be KDP-accepted format
4. File size: must be under 50MB
</context>
